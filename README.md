# Data-Structures

## How to run

### Go to src folder

```bash
cd src/
```

### Compile

```bash
make all
```

### Run

```bash
./exec
```

### Clean

```bash
make clean
```

### Or everything at once

```bash
clear && make all && ./exec && make clean
```
